SecurityTester:правила использования.

  • При запуске программы можно сразу начать выполнять полную проверку, после чего программа отобразит результат в виде диаграммы.
  • В настройках программы можно включать или выключать разделы теста, для последующего прохождения, но это необходимо делать до нажатия кнопки "Старт".
  • Правым кликом по диаграмме результата можно вызвать контекстное меню, которое позволит сохранить диаграмму в виде картинки.
  • Для выхода нажмите крестик, программа не предусматривает сохранение настроек, при последующем открытии ваши личные настройки не сохранятся.
